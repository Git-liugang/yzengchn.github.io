nginx常用命令和基本操作及问题解决

yzengchn 发表于 2018-10-12    标签： Software / Productivity

nginx常用命令：
启动nginx
./usr/local/nginx/sbin/nginx
停止nginx
nginx -s stop
重启nginx
./sbin/nginx -s reload
nginx -s reload
查看进程
ps aux | grep nginx
验证配置是否正确: nginx -t
查看Nginx的版本号：nginx -V
